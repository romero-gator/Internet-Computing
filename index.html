Assignment 1

In this assignment I was tasked with creating a portfolio website that will serve as a homepage to hold links, descriptions, and reflections related to all of my further assignments in this class, CIS 4930: Internet Computing. I was also tasked with building a fictitious web-based lesson about a topic I was interested in. This webpage had to contain meaningful content organized in a professional look and feel. The content was required to have embedded images that would contribute to the content of the page. Additionally, I had to create at least five assessment items related to the content to test the reader on. These five assessment items would link to another page dedicated to responding to the reader's input with appropriate feedback and links to the relevant material. Finally, I had to create a CSS file to organize and standardize the formatting and styling of my webpages.

Looking back at it, this was a great opportunity for me to refresh my HTML and CSS skills which have been dormant for about one year. I enjoyed messing around with new styles and seeing how they affected the look and feel of my webpages. I feel much more comfortable using HTML and CSS after completing this assignment. I often found myself spending lots of time searching for assets and researching styles rather than writing HTML or CSS. For instance, I searched for the exact images that I wanted for hours and spent tons of time looking at different fonts, sizes, colors, etc. As I become more familiar with HTML and CSS I'm sure my styling abilities will become more efficient.

    Learning Objectives

  • Refresh basic HTML and CSS skills
  • Learn how to access and use the UF servers
  • Organize and present content in a meaningful way
  • Develop effective and intuitive assessment items
  • Maintain consistent formatting and styling

I also performed unit tests to ensure my program was producing the correct results.

This assignment seemed easier than the previous ones because I had developed a good, reusable form in the previous assignment which I used here. Additionally, the algorithmic style of programming comes easier to me than the tagging and styling because I have a strong foundation with C++. Looking back at this assignment, I enjoyed coding with Javascript as it reminded me of C++ which is my favorite language to use. I liked being able to create modular code which I can reuse in the future.

Assignment 4

In this assignment I revamped the contents of assignment 2 with Bootstrap and JQuery. I used Bootsrap to make the webpage more responsive to different device sizes. I also used Bootstrap for its quick and easy styling techniques. I harnessed the power of JQuery to manipulate my webpage as the user interacted with it. This took the place of traditional Javascript. I also used JQuery to validate the form fields on my webpage. This allowed me to create a more dynamic experience for the user.

After completing this assignment, I have witnessed the power of tools like Bootstrap and JQuery and have experienced how much easier it is to develop webpages with them. I especially liked how easy it was to quickly style the page by using Bootsrap. Using JQuery was easy and simpler than plain Javascript in most cases. I really enjoyed using these tools to make the page more responsive. It had a big impact on the feel of the webpage and certainly enhanced the user experience. I look forward to using these tools in the future.

    Learning Objectives

  • Utilize Bootstrap to style the page with responsive elements
  • Utilize JQuery to create a dynamic experience
  • Validate parts of the page with JQuery
  • Learn how to develop web pages faster with Bootstrap and JQuery
